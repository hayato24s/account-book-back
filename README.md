# account-book-back

家計簿アプリのバックエンドです。  
データベースはmysqlを使用しています。  
プロントエンドは[こちら](https://github.com/hayato24s/account-book-front)

1. `yarn install`で依存パッケージをインストール
2. `yarn dev`で[localhost:8080](https://localhost:8080)にサーバーが起動します

# mysql
| Key | Value |
----|---- 
| host | localhost |
| port | 3306 |
| username | mysql |
| password | mysql |
| database | account_book |
